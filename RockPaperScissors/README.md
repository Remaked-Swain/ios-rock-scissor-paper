# 묵찌빠 프로젝트
가위바위보, 묵찌빠 게임을 구현

## 게임 규칙
* 게임 규칙 (프로그램 흐름)
    * 자연어
    1. 가위바위보를 한다. 적절치 못한 값일 경우 1번을 반복한다.
    2. 비길 경우 1번을 반복하고 아니면 승자를 결정한 뒤 다시 가위바위보를 한다. 적절치 못한 값을 받았을 경우 승자를 반대로 바꾸고 2번을 다시 수행한다.
    3. 비길 경우 최근의 승자가 최종적으로 승리하고, 아닌 경우 2번을 반복한다.
